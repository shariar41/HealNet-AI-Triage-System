# HealNet AI Triage System

## Overview
HealNet is a multimodal AI system that combines symptom analysis, X-ray imaging, and reinforcement learning to provide intelligent healthcare triage decisions.

## Features
- Symptom-based prediction (ML)
- X-ray pneumonia detection (Deep Learning)
- Reinforcement Learning decision system
- Safety-aware triage logic

## Tech Stack
- Python
- PyTorch
- Scikit-learn
- Streamlit

## How to Run

```bash
pip install -r requirements.txt
streamlit run app/app.py
